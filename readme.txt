Web-Based Information Management System for Hindu Buddhist Cultural Assosiation

Language 
PHP

DataBase 
MySQL

Installation
Copy the folder that containig source code.
Open xampp control panel
Start Apache and MySQL
Create database in name hbcaims and import file hbcaims.sql
Change username password of databse in dbconnection.php
Open Browser and type http:\\localhost\HinduBuddhistCulturalAssosiation 

Requirments

Apache 2.x
PHP version 7.2.1
MySQL
